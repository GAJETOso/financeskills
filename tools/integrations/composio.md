# Composio Integration Guide

Composio provides a unified MCP (Model Context Protocol) layer for tools that require OAuth or have complex authentication (HubSpot, Salesforce, QuickBooks, Xero).

## 🏛️ Financial Tool Mapping
- **Accounting**: QuickBooks, Xero, FreshBooks.
- **CRM**: Salesforce, HubSpot (Finance/Deal data).
- **Communication**: Slack (Approval workflows).

## 🚀 Setup
1. Get an API key from [Composio](https://composio.dev).
2. Install the Composio MCP server.
3. Authenticate specific tools via the Composio dashboard.
