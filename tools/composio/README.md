# Composio for Finance

Mapping financial tools and accounting suites to the Composio MCP layer.

## 🏛️ Strategy
We use Composio for tools that handle sensitive OAuth flows (e.g., QuickBooks) to provide a secure, standardized interface for AI agents.

## 🛠️ Toolkits
- **`accounting-suite`**: QuickBooks, Xero.
- **`banking-suite`**: Mercury, Brex (via Composio).
- **`crm-deals`**: Salesforce, HubSpot.
