# Finance CLI Tools

Zero-dependency Node.js scripts for interacting with financial APIs and processing data.

## 🚀 Usage
```bash
node tools/clis/<tool>.js <command> --args
```

## 🛠️ Available Tools
- **`etherscan.js`**: Fetch on-chain transaction logs.
- **`conversion.js`**: Live currency conversion using public APIs.
- **`vat-verify.js`**: Verify EU VAT numbers via VIES.
