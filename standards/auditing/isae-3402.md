# ISAE 3402: Service Organisation Controls (SOC)

## What It Addresses
Provides requirements for practitioners reporting on controls at a service organisation (e.g., payroll processor, cloud provider, custodian) that are relevant to user entities' financial reporting. Equivalent to SSAE 18 (SOC 1) in the US.

## Report Types
- **Type 1**: Describes the service organisation's system and suitability of controls at a specific date (design only)
- **Type 2**: Describes system and tests the operating effectiveness of controls over a period (typically 6–12 months) — required by most auditors of user entities

## Scope
Any service organisation that processes transactions or holds assets on behalf of user entities and where those activities are relevant to the user entity's financial reporting ICFR.

## Key Requirements
- **Inclusive vs. carve-out methods**: How subservice organisations are handled
- **Complementary user entity controls (CUECs)**: Controls that must be in place at user entities for the service organisation's controls to be effective
- **Auditor's report contents**: Description of system; management's assertion; practitioner's opinion on fairness of description and suitability/effectiveness of controls

## User Auditor Linkage (ISA 402)
User entity auditors use ISAE 3402 Type 2 reports to obtain evidence about controls at service organisations relevant to their ISA 330 responses. A clean Type 2 report reduces required substantive testing at the user entity.
