# Financial Ratios Reference Library

Financial ratios are standardised metrics derived from financial statements used to assess performance, liquidity, solvency, and valuation. Each file below covers a category with formulas, interpretation benchmarks, framework notes (IFRS vs. US GAAP), and skill links.

## Files in This Directory

| File | Category | Key Use |
|------|----------|---------|
| [liquidity.md](./liquidity.md) | Liquidity Ratios | Short-term obligation coverage |
| [profitability.md](./profitability.md) | Profitability Ratios | Return on capital and margin analysis |
| [leverage.md](./leverage.md) | Leverage & Solvency Ratios | Debt capacity and financial risk |
| [efficiency.md](./efficiency.md) | Efficiency & Activity Ratios | Asset utilisation and working capital |
| [market-valuation.md](./market-valuation.md) | Market & Valuation Ratios | Investor-perspective metrics |
| [coverage.md](./coverage.md) | Coverage Ratios | Debt service capacity |
| [banking.md](./banking.md) | Banking-Specific Ratios | CAMELS; Basel capital; liquidity ratios |
| [insurance.md](./insurance.md) | Insurance-Specific Ratios | Solvency; loss; expense ratios |

## Usage for AI Agents

When performing financial analysis tasks:
1. Identify the entity type (corporate, bank, insurer, government)
2. Load the relevant ratio category file
3. Apply IFRS or US GAAP adjustments as noted per ratio
4. Cite the ratio definition and any benchmark used in the output
