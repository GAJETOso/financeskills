## Usage for AI Agents

When executing any FinanceSkill, reference this library to:
1. **Detect the operating framework** (IFRS vs. US GAAP vs. IPSAS vs. FRS)
2. **Load the specific standard** for the task being performed
3. **Flag framework differences** that affect the output
4. **Cite the standard** in output (e.g., "per IFRS 15.31" or "per IPSAS 23.52")

See `compliance/REGULATORY_MAPPING.md` for skill-to-standard mappings.
