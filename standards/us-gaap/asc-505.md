# ASC-505: Equity

## What It Addresses
Accounting for equity transactions including issuance, repurchases, dividends, convertible instruments, and changes in equity structure.

## Key Requirements
- Treasury stock: cost method (most common) or par value/stated capital method
- Dividends: recognised as reduction in retained earnings on declaration date (declaration creates liability if distributed later)
- Stock splits: no journal entry for memorandum splits; forward/reverse splits adjust par value
- Share repurchases: reduce equity; no gain/loss recognised on treasury stock transactions
- Quasi-reorganisation: eliminate deficit and write assets down to fair value under specific circumstances

## Key Disclosures
- Number of shares authorised, issued, outstanding for each class
- Rights and restrictions for each class
- Dividends declared per share; aggregate dividends
- Treasury shares: number and cost

## Key Differences from IFRS
IAS 32 governs equity classification; IAS 1 covers equity presentation. IFRS requires OCI components to be identified as items that will/will not be reclassified to P&L. US GAAP has accumulated other comprehensive income (AOCI) presentation; similar in concept.
