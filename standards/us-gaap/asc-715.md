# ASC-715: Compensation — Retirement Benefits

## What It Addresses
Accounting for defined benefit and defined contribution pension plans and other postretirement benefit plans.

## Key Requirements
- Defined contribution plans: expense = contribution; no actuarial complexity
- Defined benefit plans: projected benefit obligation (PBO) using projected unit credit method
- Funded status: recognised on balance sheet (asset or liability = plan assets − PBO)
- Net periodic pension cost: service cost + interest cost − expected return on plan assets ± amortisation of prior service cost ± amortisation of actuarial gains/losses
- Corridor approach: actuarial gains/losses amortised to P&L if outside 10% corridor (vs. IFRS immediate OCI recognition)
- Other postretirement benefits (OPEB): similar accounting framework

## Key Disclosures
- Funded status reconciliation
- Components of net periodic benefit cost
- Weighted average assumptions (discount rate, expected return on assets, compensation increase)
- Plan asset allocation and fair value by level
- Expected future benefit payments for next 10 years

## Key Differences from IFRS
IAS 19: actuarial gains/losses recognised immediately in OCI with no recycling. US GAAP uses corridor approach — amortise to P&L. Expected return on assets: IAS 19 eliminated this concept; uses net interest (rate applied to net funded status).
