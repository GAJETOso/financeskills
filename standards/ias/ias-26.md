# IAS-26: Accounting and Reporting by Retirement Benefit Plans

## What It Addresses
Addresses accounting and reporting by retirement benefit plans to all participants as a group.

## Scope
Financial statements of retirement benefit plans (pension funds), not the financial statements of employers. Both defined contribution and defined benefit plans.

## Key Requirements
- Defined contribution plans: report net assets available for benefits; describe funding policy
- Defined benefit plans: report net assets available for benefits; actuarial present value of promised retirement benefits (distinguishing vested from non-vested)
- Plan assets: reported at fair value (market value for marketable securities)
- Actuarial present value disclosure: based on projected salary levels or current salary levels — must state which and be consistent

## Key Disclosures
- Statement of net assets available for benefits
- Statement of changes in net assets available for benefits
- Description of funding policy; significant accounting policies
- Description of the plan; investment policies

## Key Differences from US GAAP
US GAAP (ASC 960/962/965): more detailed standards for each plan type. Similar asset valuation at fair value approach.
