# ESG Reporting Framework Landscape

## Convergence Map

```
TCFD (2017) ──────────────────────► IFRS S1/S2 (ISSB, 2023) ◄── SASB metrics
                                           │
                          ┌────────────────┼────────────────┐
                          ▼                ▼                ▼
                   SEC Climate Rule    CSRD/ESRS E1    CBN / SEC Nigeria
                     (US listed)        (EU large)      (Nigerian entities)
                          
GRI Standards ────────────────────► CSRD/ESRS (double materiality)
                                           
TNFD (2023) ──────────────────────► CSRD/ESRS E4 (Biodiversity)
CDP ───────────────────────────────► Aligned with TCFD/IFRS S2/GRI
```

## Which Framework Applies to Which Entity?

| Entity Type | Primary Framework | Secondary / Aligned |
|-------------|------------------|---------------------|
| IFRS-adopting listed (non-EU) | IFRS S1 + S2 | GRI (impact); CDP; SASB |
| EU large company | CSRD / ESRS | IFRS S2 (convergence ongoing) |
| US listed (SEC) | SEC climate rule | IFRS S2 (voluntary); SASB |
| Nigerian listed (NGX) | SEC Nigeria (voluntary TCFD) | GRI; IFRS S2 (forthcoming) |
| Nigerian bank | CBN climate guidelines (TCFD) | IFRS S2; GRI |
| Nigerian oil and gas | IFRS S2; GRI 11 (Oil & Gas) | TCFD; CDP |
| Global supply chain | CSRD value chain requests | GRI; CDP |

## Double Materiality vs. Financial Materiality

| Framework | Materiality Concept |
|-----------|--------------------| 
| IFRS S1/S2 | Financial materiality (investor-focused) |
| GRI | Impact materiality (impact on world) |
| CSRD/ESRS | Double materiality (both) |
| TCFD / SEC | Financial materiality |
| CDP | Financial + impact (CDP scoring rewards both) |

## Assurance Requirements

| Framework | Assurance |
|-----------|-----------|
| CSRD/ESRS | Mandatory limited assurance (→ reasonable) |
| IFRS S2 | Not mandated at ISSB level; jurisdictional |
| SEC | Required (limited → reasonable, phased) |
| GRI | Voluntary; encouraged |
| CDP | Not required; verification data requested |
