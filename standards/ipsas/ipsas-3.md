# IPSAS-3: Accounting Policies, Changes in Accounting Estimates and Errors

## What It Addresses
Prescribes criteria for selecting accounting policies and treatment of changes and error corrections for PSEs.

## Scope
All PSEs applying accrual-based IPSAS.

## Key Requirements
- Hierarchy for selecting policies when no IPSAS applies
- Changes in accounting policy: retrospective application
- Changes in estimates: prospective application
- Error corrections: retrospective restatement

## Key Disclosures
- Nature and reason for change in accounting policy
- Effect on current and prior periods
- Description and amount of corrections of errors

## Relationship to Private Sector Standard
Adapted from IAS 8. Substantially similar principles applied to public sector context.
