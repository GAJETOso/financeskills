# IPSAS-16: Investment Property

## What It Addresses
Prescribes accounting for investment property held by PSEs (e.g., commercial properties held for rental income by government entities).

## Scope
PSEs holding property to earn rentals or for capital appreciation.

## Key Requirements
- Cost model or fair value model
- Fair value model: changes in fair value recognised in surplus/deficit
- Properties held for social purposes vs. for commercial return: different classification

## Key Disclosures
- Measurement model; fair value if cost model used
- Rental income; direct operating expenses
- Contractual obligations to develop or repair investment property

## Relationship to Private Sector Standard
Adapted from IAS 40. Same choice of cost or fair value model.
