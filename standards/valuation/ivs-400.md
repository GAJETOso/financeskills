# IVS 400: Real Property Valuation

## What It Addresses
Valuation of interests in real property — freehold, leasehold, investment property, development land. Relevant for IAS 16 (revaluation model), IAS 40 (investment property FV model), IFRS 16 (lease term assessment).

## Key Methods
- **Sales Comparison (Market Approach)**: Comparable recent transactions; adjust for location, condition, size, date
- **Investment Method (Income Approach)**: Capitalise passing rent or ERV; equivalent yield; DCF for reversionary properties
- **Depreciated Replacement Cost (DRC)**: Specialised properties with no market evidence; used in IPSAS 17 for infrastructure and government assets
- **Residual Method**: Development properties (IVS 410) — GDV minus development costs minus developer's profit

## IFRS 40 Context
Entities using the fair value model for investment property must use IVS 400-consistent methodology. Annual FV change recognised in profit or loss (IAS 40.35). External valuer engagement recommended for material properties.

## Nigeria Context
ESVARBON-registered valuers required for statutory property valuations in Nigeria. Lagos, Abuja, Port Harcourt commercial yields: 6–9% typical; industrial: 8–12%.
