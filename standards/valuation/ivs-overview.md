# IVS Framework

## What It Addresses
The IVS establish globally consistent standards for valuation of assets, businesses, and financial instruments. Used by valuers, auditors assessing fair values under IFRS 13/ASC 820, M&A advisors, and lenders.

## Structure
- **IVS Framework**: Underlying concepts — credibility, competence, objectivity
- **General Standards (IVS 101–105)**: Apply to all valuations regardless of asset type
- **Asset Standards (IVS 200–500)**: Asset-class-specific requirements

## Key Concepts
- **Bases of Value**: Market Value (most common); Investment Value (entity-specific synergies); Fair Value (IFRS 13); Liquidation Value; Synergistic Value
- **Valuation Approaches**: Market Approach; Income Approach; Cost Approach — all three must be considered
- **Highest and Best Use (HBU)**: For real property and some assets — the use that maximises value and is legally permissible, financially feasible, and physically possible
- **Market Participant**: Valuation based on assumptions of hypothetical market participants, not entity-specific assumptions (aligns with IFRS 13 exit price concept)

## IFRS 13 Linkage
IVS is highly aligned with IFRS 13 Fair Value Measurement. IFRS 13 defines fair value using an exit price concept with a Level 1/2/3 hierarchy; IVS provides the methodological detail for implementing valuations that comply with IFRS 13.

## Nigeria / Africa Context
- ESVARBON (Estate Surveyors and Valuers Registration Board of Nigeria) — regulates valuers
- Property valuations for IFRS 40 (Investment Property fair value model) and IFRS 16 (lease terms) commonly use IVS-aligned methodology
- Oil field valuations (reserve-based lending, impairment) draw on IVS income approach
