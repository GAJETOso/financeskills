# Contributing to FinanceSkills

Welcome! We are excited that you want to contribute to the FinanceSkills ecosystem.

## How to Contribute
1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: Use a descriptive name for your branch (e.g., `feat/new-tax-skill`).
3.  **Add your Skill**: Follow the `SKILL.md` template in the `skills/` directory.
4.  **Validate**: Run `./validate-skills.sh` to ensure your skill meets the technical requirements.
5.  **Submit a PR**: Provide a clear description of your changes.

## Skill Guidelines
- Every skill must have a clear `When to use` section.
- Steps must be actionable for an AI agent.
- Include a `verification` section to audit the results.

## Code of Conduct
Please be respectful and professional in all interactions.
