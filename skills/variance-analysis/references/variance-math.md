# Manufacturing Variance Formulas

Quick-reference for calculating standard costing variances in a production environment.

## 🏛️ Material Variances

### Material Price Variance (MPV)
`MPV = (Standard Price - Actual Price) * Actual Quantity`
- **Favorable**: If Actual Price < Standard Price.

### Material Usage Variance (MUV)
`MUV = (Standard Quantity - Actual Quantity) * Standard Price`
- **Favorable**: If Actual Quantity < Standard Quantity.

---

## 🏛️ Labor Variances

### Labor Rate Variance (LRV)
`LRV = (Standard Rate - Actual Rate) * Actual Hours`
- **Favorable**: If Actual Rate < Standard Rate.

### Labor Efficiency Variance (LEV)
`LEV = (Standard Hours - Actual Hours) * Standard Rate`
- **Favorable**: If Actual Hours < Standard Hours.

---

## 🔍 Overhead Variances
- **Variable Overhead Spending**: `(Standard Rate - Actual Rate) * Actual Hours`.
- **Variable Overhead Efficiency**: `(Standard Hours - Actual Hours) * Standard Rate`.
