# The Venture Capital Method

## Core Logic
Work backwards from exit:
1. **Exit value** = projected exit-year metric × comparable multiple (e.g., year-5 revenue 50M × 6x EV/Revenue = 300M).
2. **Required future stake** = investment × (1 + target return)^n / exit value. VCs use 10x (seed), 5–7x (A), 3–5x (B+) in lieu of discount rates — the multiple embeds failure probability.
3. **Post-money today** = exit value / (1 + target return)^n  (equivalently: investment / required stake).
4. **Pre-money** = post-money − investment.

## Worked Example
2M seed investment, 10x target, exit value 300M in 6 years:
- Required exit proceeds = 20M → required stake = 20/300 = 6.7% at exit.
- **Dilution adjustment**: future rounds will dilute the seed holder. If expecting 40% cumulative dilution, required today's stake = 6.7% / (1 − 0.40) = 11.1%.
- Post-money = 2M / 11.1% = 18M; pre-money = 16M.

## Dilution Adjustment (the step everyone forgets)
`Required current % = required exit % / (1 − expected cumulative dilution)`
Typical per-round dilution 15–25%; seed-to-exit cumulative often 40–60%. Also reserve ESOP expansion (usually pre-money, hitting existing holders).

## Cross-Checks
- Implied ownership vs. market norms (seed 10–20%, A 15–25%).
- Exit value sanity: does it require implausible market share?
- Compare against scorecard/Berkus (pre-revenue) and comparable round pricing.
