# Spending Categories Guide

Standard taxonomy for classifying personal expenses. Use consistently across months for valid trend analysis.

## Needs (Essential)
| Category | Examples |
|---|---|
| Housing | Rent, mortgage, property tax, HOA |
| Utilities | Electricity, gas, water, internet, phone |
| Groceries | Supermarket food and household basics |
| Transport | Fuel, public transit, car insurance, maintenance |
| Insurance | Health, life, renters/home |
| Minimum debt payments | Credit card minimums, student loan minimums |
| Childcare / education | School fees, daycare |

## Wants (Discretionary)
| Category | Examples |
|---|---|
| Dining & takeaway | Restaurants, cafes, food delivery |
| Entertainment | Streaming, cinema, concerts, sports |
| Subscriptions | Software, magazines, gym (if non-essential) |
| Clothing | Non-essential apparel and accessories |
| Hobbies | Sports equipment, crafts, gaming |
| Travel | Holidays, weekend trips |
| Personal care | Haircuts, cosmetics above basics |

## Savings & Debt Payoff
| Category | Examples |
|---|---|
| Emergency fund | High-yield savings account contributions |
| Retirement | 401(k), IRA, pension contributions |
| Investments | Brokerage, ISA, index funds |
| Extra debt repayment | Above-minimum payments on loans/cards |
| Sinking funds | Car replacement, home repairs, annual expenses |

## Classification Tips
- **Internet**: Needs if required for work/study; Wants if purely recreational.
- **Gym membership**: Needs if medically required; Wants otherwise.
- **Life insurance**: Always a Need.
- **Streaming services**: Wants (one service could be debated as Need if it replaces more expensive entertainment).
