# PMI Earned Value Management Standards

The global standard for project performance measurement.

## 🏛️ Core Metrics

### 1. The Triad
- **Planned Value (PV)**: Budgeted cost for the work scheduled to be completed.
- **Actual Cost (AC)**: Total cost incurred for the work performed.
- **Earned Value (EV)**: Budgeted cost for the work actually performed.

### 2. Performance Indices
- **CPI (Cost Performance Index)**: `EV / AC`. 
  - > 1.0: Under budget.
  - < 1.0: Over budget.
- **SPI (Schedule Performance Index)**: `EV / PV`.
  - > 1.0: Ahead of schedule.
  - < 1.0: Behind schedule.

---

## 🔍 Forecasting (EAC)
**Estimate at Completion (EAC)**: The expected total cost of the project.
- **Formula 1 (Current rate continues)**: `BAC / CPI`.
- **Formula 2 (Remaining work at budgeted rate)**: `AC + (BAC - EV)`.
