# Scenario Analysis: Modeling the Unknown

How to structure "What-If" scenarios to test the resilience of a financial plan.

## 🏛️ Types of Scenarios

### 1. The Base Case
The most likely outcome based on current trends and management's stated strategy.

### 2. The Bull Case (Upside)
What happens if everything goes right? (e.g., 2x market growth, successful product launch).

### 3. The Bear Case (Downside)
What happens if things go wrong? (e.g., major competitor enters, recession, supply chain failure).

---

## 🔍 How to Build a Scenario Matrix
1.  **Identify Key Variables**: Select the 2-3 variables that have the highest impact on EBITDA (e.g., Sales Volume, Labor Cost).
2.  **Define Thresholds**: Use +/- 10% or historical worst-case data points.
3.  **Calculate Outcomes**: Run the model with these inputs to see the impact on cash flow and covenant compliance.
